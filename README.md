# 🏦 CreditWise Loan Approval System

A machine learning web app that predicts whether a loan will be approved or not based on applicant details.

## 🚀 Live Demo
Coming soon...

## 📌 Project Overview
This project uses machine learning to predict loan approvals based on factors like credit score, income, savings, DTI ratio and more.

## 🛠️ Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn
- Streamlit
- Plotly

## 🤖 Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes ✅ (Best Model - 86.5% Accuracy)

## 📊 Features
- Interactive loan prediction form
- Loan Eligibility Gauge Chart
- Approved / Rejected result with score

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   pip install streamlit pandas numpy scikit-learn plotly
3. Run the app:
   streamlit run app.py

## 📁 Dataset
- 1000 applicant records
- Features: Income, Credit Score, DTI Ratio, Savings, Collateral and more
